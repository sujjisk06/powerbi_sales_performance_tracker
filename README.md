Project Overview
This project is a Sales Performance Tracker dashboard developed using Power BI.
It analyzes sales data to provide insights into revenue, profit, and performance across regions, products, and time periods, helping support data-driven business decisions.

Business Objectives
Monitor overall sales and profit performance
Identify top and bottom performing products
Analyze sales trends by region and time
Track key performance indicators (KPIs)
Tools and Technologies
Power BI
Power Query
DAX
Excel
Key KPIs
Total Sales
Total Profit
Profit Margin
Sales Growth
Top Products
Regional Performance
Data Cleaning and Transformation
The following steps were performed using Power Query:

Removed null and duplicate records
Removed unnecessary columns
Standardized column names and data types
Created calculated columns and measures using DAX
Established relationships between tables
Dashboard Features
Interactive slicers for Year, Region, and Product
Sales and profit trend analysis
Product-wise and region-wise performance comparison
Drill-down and filtering for detailed analysis
